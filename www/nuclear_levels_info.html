Описание опыта:
В данном опыте у пользователя есть возможность выбрать любое количество спектральных серий водорода для показа на диаграмме энергетических уровней.
Также пользователь может рассчитать, сколько энергии будет выделено или поглощено при переходе електрона между энергетическими уровнями. Для этого нужно нажать на энергетический уровень, на который вы хотите переместить электрон.

Теория:
Энергетические переходы:
При переходе между энергетическими уровнями атома водорода электрон поглощает или выделяет энергию. Поглощение происходит при удалении электрона от ядра. Выделение, соответственно, при приближении электрона к ядру.
E = -13.6 * (1/n^2 - 1/n’^2) [эВ], где
n - номер текущего уровня,
n’ - номер уровня, на который
осуществляется переход.

Спектральные серии:
Спектральные серии водорода — набор спектральных серий, составляющих спектр атома водорода.
Спектральная серия — набор спектральных линий, которые получаются при переходе электронов с любого из вышележащих термов на один нижележащий, являющийся основным для данной серии.
Спектральное излучение происходит, когда электрон переходит из более высокого энергетического состояния в более низкое. Энергия излучённого фотона постоянна (так как энергия каждого состояния всегда одинакова) и соответствует разнице двух состояний.
