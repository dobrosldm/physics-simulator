Дифракция

В нашем опыте пользователь может наблюдать за данным явлением. Также пользователь может изменять размер препятствия и в соответствие с этим будет меняться картина, но светлое пятно Пуассона будет всегда в центре. Для разработки этого были использован язык разметки html, каскадные таблицы стилей, а также язык программирования javascript и элемент html 5 для создания 2d рисунков (canvas).
